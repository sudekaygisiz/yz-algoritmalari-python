# 🧠 Python ile Makine Öğrenmesi Algoritmaları ve Uygulamaları

Bu repository, makine öğrenmesi temelleri üzerine aldığım eğitim boyunca öğrendiğim algoritmaların teorik altyapılarını ve Python (Scikit-Learn) ile yapılmış pratik uygulamalarını içermektedir. Repoda, temel regresyon modellerinden karmaşık kümeleme ve boyut indirgeme algoritmalarına kadar geniş bir yelpazede kod örnekleri bulunmaktadır.

## İçerikte Neler Var?

Çalışmalarım temel olarak iki ana kategoriye ayrılmaktadır:

### 1. Gözetimli Öğrenme (Supervised Learning)
* **Regresyon Modelleri:** Linear Regression, Polynomial Regression, Logistic Regression.
* **Sınıflandırma Algoritmaları:** K-Nearest Neighbors (KNN), Naive Bayes, Support Vector Machines (SVM).
* **Ağaç Bazlı Algoritmalar:** Decision Trees, Ensemble Methods ve Random Forests.

### 2. Gözetimsiz Öğrenme (Unsupervised Learning)
* **Kümeleme (Clustering):** K-Means, Hierarchical Clustering ve DBSCAN.
* **Boyut İndirgeme:** Principal Component Analysis (PCA) ve PCA ile Yüz Tanıma (Face Recognition) uygulamaları.

## Kullanılan Teknikler ve Kavramlar
Algoritmaların yanı sıra, modellerin performansını artırmak ve doğruluğunu ölçmek için uyguladığım temel veri bilimi teknikleri:
* Gradient Descent Optimizasyonu
* Overfitting (Aşırı Öğrenme) Önleme & Regularization
* Cross Validation & Confusion Matrix Analizi
* Grid Search ile Hiperparametre Optimizasyonu (Parameter Tuning)
* Pipeline Method ve Data Leakage Önlemleri

## Teknolojiler
* Python
* Scikit-Learn
